## 第一批：代码审查 Q3（可在 15 分钟内完成）

### Q3 — 标签优先级向量化

**文件**: `generator.py`  
**位置**: 病害循环末尾的逐点优先级回退  
**当前**:
```python
for i in np.where(changed_mask)[0]:
    old_pri = LABEL_PRIORITY.get(int(old_labels[i]), 0)
    new_pri = LABEL_PRIORITY.get(int(labels[i]), 0)
    if old_pri > new_pri:
        labels[i] = old_labels[i]
```
**改为**:
```python
# 预构建查找表
max_label = max(LABEL_PRIORITY.keys()) + 1
priority_lut = np.zeros(max_label, dtype=np.int32)
for k, v in LABEL_PRIORITY.items():
    priority_lut[k] = v
# 向量化裁决
old_pri = priority_lut[old_labels[changed_mask]]
new_pri = priority_lut[labels[changed_mask]]
restore = new_pri < old_pri
labels[changed_mask] = np.where(restore, old_labels[changed_mask], labels[changed_mask])
```
