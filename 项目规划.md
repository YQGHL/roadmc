#### 1.1.1 路面宏观轮廓生成

```python
def generate_road_surface(width: float, length: float, grid_res: float,
                         pavement_type: str = 'asphalt',
                         roughness_class: str = 'A') -> Tuple[np.ndarray, np.ndarray]:
```

- 采用 **ISO 8608 路面功率谱密度（PSD）** 生成纵向轮廓：
    $$ G_d(n) = G_d(n_0) \left( \frac{n}{n_0} \right)^{-w} $$
  其中 $n_0 = 0.1 \ \text{cycle/m}$，$w=2$ 取决于粗糙度等级（A-B-C..）。
- 用逆傅里叶变换生成纵向高度序列，再通过横向指数关联扩展为二维表面，公式：
    $$ h(x,y) = \sum_{i,j} \sqrt{G_d(n_i) G_d(n_j)} \cos(2\pi n_i x + \phi_{ij}) \cos(2\pi n_j y + \psi_{ij}) $$
  其中 $\phi, \psi$ 为随机相位。
- 同时计算每个点的解析法向（通过数值差分）。

- 使用 **分数布朗运动（fBm）** 将高频纹理叠加在法线方向：$z' = z + A \sum_k fBm_k$。

#### 1.1.3 裂缝生成（沥青路面：纵向/横向裂缝，龟裂，块状裂缝）

```python
def add_crack(points, labels, crack_type, params):
```

- **纵向/横向裂缝**：骨架路径由三次 Bézier 曲线定义，并叠加分形扰动（Perlin 噪声调制）。
- 裂缝开口宽度 $w$ 沿路径由对数正态分布采样，符合 $w \sim \text{Lognormal}(\mu, \sigma)$。
- 深度剖面使用 **V 形槽** 或高斯槽：$d(t) = d_{\max} \exp\left( - (t / \lambda)^p \right)$，$p$ 控制形状，$\lambda$ 与宽度相关。
- **龟裂（疲劳裂缝）**：通过空间泊松点过程生成裂缝网络节点，然后用 Voronoi 图生成多边形边界，沿边界叠加裂缝基元。根据连通域密度和宽度区分轻重（轻：主要块度>0.5m，重：<0.2m）。
- **块状裂缝**：类似龟裂但块度更大（>1m），采用低密度 Voronoi 图。
- 标签：龟裂轻=1，重=2；块状裂缝轻=3，重=4；纵向裂缝轻=5，重=6；横向裂缝轻=7，重=8。重度裂缝开口更宽、深度更大、边缘剥落更严重。

#### 1.1.4 坑槽生成

```python

```

- 使用旋转超椭圆凹陷：
    $$ z(r) = -d_{\max} \left[ 1 - \left( \frac{r}{R} \right)^\beta \right]^{1/\beta} $$
  $\beta=2$ 为椭球，$\beta>2$ 产生平底坑（重度坑槽）。
- 边缘剥落使用随机半径侵蚀 + 泊松采样的小坑。
- 轻度坑槽（标签9）：$d_{\max} \le 25\text{mm}$；重度坑槽（标签10）：$d_{\max} > 25\text{mm}$。
