# 代数几何与泛函分析在 RoadMC 项目中的应用研究方向

## 一、代数几何：参数量化的病害形态建模

### 1.1 裂缝骨架的平面代数曲线

- **表示**：用多项式 $f(x,y)=0$ 的零点集定义裂缝路径。
- **优势**：分支点奇异性可由 $\delta$-不变量、Milnor 数定量描述。
- **应用**：龟裂网络 → **奇异代数曲线的分支**，通过随机多项式系数生成类网状裂缝，比 Voronoi 图更符合应力分叉的力学机理。

### 1.2 坑槽与变形的代数曲面

- **表示**：$z - g(x,y) = 0$，$g$ 为非负多项式。
- **优势**：多项式次数直接控制几何复杂度（平整→破损的连续演化）。
- **应用**：在 `primitives.py` 中构造 `AlgebraicDamage` 基类，统一多种路面缺损的参数化生成。

## 二、泛函分析：网络稳定性与分布对齐

### 2.1 mHC 的算子谱理论

- **观点**：Transformer 块 → Hilbert 空间上的算子，mHC 的 H 矩阵 → Markov 算子。
- **理论工具**：算子的数值域、Krein-Milman 定理（Birkhoff 多胞体的极点为置换矩阵）。
- **效果**：证明深度 mHC 网络保持在有界 $L^2$ 范数球内，并为连接矩阵的稀疏化提供理论基础。

### 2.2 GAN 域适应的积分概率度量

- **现有**：WGAN-GP 基于 Lipschitz 函数空间。
- **泛函方法**：引入 RKHS 下的 **最大均值差异（MMD）**，使用正定核（如基于倒角距离的拉普拉斯核）。
- **优势**：优化曲面更光滑，避免模式坍塌，将风格迁移形式化为分布嵌入的匹配。

### 2.3 基于谱方法的点云注意力

- **关联**：点云自注意力 ↔ 非局部算子；图示：邻接矩阵的图拉普拉斯算子。
- **方法**：计算几何图的图傅里叶基（Lanczos 近似前 $k$ 个特征向量），将注意力限制在低频子空间。
- **效果**：天然捕获沉陷、车辙等全局低频变形，同时滤除纹理噪声。
